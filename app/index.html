Get the desktop app
✕
The desktop version lives as a small tab on the edge of your screen — always accessible, no browser needed.
01
Download Node.js
The desktop app runs on Electron, which requires Node.js. Download and install it first — it's a one-time step.
Download Node.js →
Choose the LTS version. Run the installer and follow the prompts.
02
Download the app files
Download the dig systems desktop package and unzip it somewhere you'll remember — your Documents folder works well.
Download dig systems →
Unzip the folder after downloading. Don't delete it — the app runs from this folder.
03
Open Terminal and install
Open Terminal (press ⌘ Space, type Terminal, press Enter). Then type cd  with a space after it, drag your unzipped dig-systems folder into the Terminal window, and press Enter.
cd ~/Documents/dig-systems
Then run this to install dependencies (takes about a minute):
npm install
You only need to do this once.
04
Launch the app
Still in Terminal, run:
npm start
A small tab will appear on the right edge of your screen. Click it to open dig systems. From now on you can launch it the same way any time.
If macOS blocks the app, go to System Settings → Privacy & Security and click "Open Anyway".
